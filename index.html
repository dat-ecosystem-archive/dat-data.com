dat is an open source tool that enables the sharing of large datasets, the goal being a collaboration flow similar to what git offers for source code.

It isn't quite ready for prime time yet, but if you want to play around with the pre-alpha version please check out these slides or read the readme of the dat repository on GitHub.

Request some stickers and we will mail you some for free! You can keep them or give them out to people who support open data.
