dat is an open source tool that enables the sharing of large datasets, the goal being a collaboration flow similar to what git offers for source code. As a team we have a bias towards supporting scientific + research data use cases.

What we're building isn't quite ready for prime time yet, but if you want to play around with the pre-alpha version of dat please check out these slides or read the readme of the dat repository on GitHub.

Request some stickers and we will mail you some for free! You can keep them or give them out to people who support open data.

The dat project is officially a project of the US Open Data Institute, a not-for-profit organization based in the United States, but we work towards sharing data across the galaxy.
